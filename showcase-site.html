Build Step

Three steps. Two minutes. All the sounds.

01
Clone and Install
Pure Python + vanilla JS. No build step. No node_modules.
# Clone the repository git clone https://github.com/user/quick-ping-2.git pip3 install flask flask-cors
02
Hook into Claude Code
Add Quick-Ping to your Claude Code hooks. One config block, three hook points.
# Add to ~/.claude/settings.json "PreToolUse", "PostToolUse", "Stop"
03
Launch and Code
Open the Control Panel. Pick a collection. Start coding.
./launch-control-panel.sh # Open localhost:5050
